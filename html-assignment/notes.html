5 HTML tags I have learnt in Web Design Foundations

ul/ol
Unordered/Ordered List. Holds a series of items. Items in an Unordered List have no order, while items in an Ordered List have a specified order to be followed
dl
Description List. Holds entries like a normal list (dt, description term), but also holds information about those entries (dd, description)
a
HTML Anchor. Allows linking to anything (website, specific page, picture, anything available on the web)
table
Creates a table. Filled with information via tr (table row) and td (table data)
img
Inserts an image onto the page. Requires src (image source) and alt (alt text, accessibility feature) to work
